Local PubMed Search Guide / 本地 PubMed 检索指南

Use the web interface for PubMed-style literature retrieval, or call the API directly for backend and batch workflows.

可以使用网页进行类似 PubMed 的文献检索，也可以直接调用 API 支持后端、批量和导出流程。

Navigation

Start in the main search box, choose a mode, then refine with the left facets. Use Advanced for author, journal, MeSH, chemical, keyword, language, publication type, and year limits.

  1. Run a broad keyword query.
  2. Click MeSH, year, journal, source, or status facets to narrow the set.
  3. Open a title for full metadata, abstract sections, IDs, and retraction links.
  4. Export the current result set as compressed JSON when the query is stable.

操作路径

先使用主检索框，选择检索模式，再通过左侧分面筛选收窄结果。高级检索适合限定作者、期刊、MeSH、化合物、关键词、语言、出版类型和年份。

  1. 先运行一个较宽的关键词检索。
  2. 点击 MeSH、年份、期刊、数据来源或文献状态分面收窄集合。
  3. 点击题名打开完整元数据、结构化摘要、标识符和撤稿关系。
  4. 检索式稳定后，将当前结果导出为压缩 JSON。

Search Syntax

The main search box accepts free text and common PubMed-style field tags.

  • single nucleotide polymorphism[Title]
  • Vashist[Author]
  • Cell Oncol[Journal]
  • Esophageal Neoplasms[MeSH]
  • "Breast Neoplasms"[MeSH] for exact descriptor matching
  • GNAS[Chemical]
  • 2010:2012[Year]

检索语法

主检索框支持自由文本，也支持常见 PubMed 风格字段标签。

  • single nucleotide polymorphism[Title]
  • Vashist[Author]
  • Cell Oncol[Journal]
  • Esophageal Neoplasms[MeSH]
  • "Breast Neoplasms"[MeSH] 表示精确匹配该 MeSH 主题词
  • GNAS[Chemical]
  • 2010:2012[Year]

Keyword Tips

  • Use 2-4 core English terms first; add field tags after seeing the first result set.
  • For diseases and standard biomedical concepts, prefer exact MeSH when possible: "Breast Neoplasms"[MeSH].
  • For variants, combine synonyms in broad mode, then refine by facets. Example: breast cancer neoplasm carcinoma.
  • Use title constraints for high precision: trastuzumab resistance[Title].
  • Use identifiers directly for lookup-oriented retrieval: PMID, DOI, PMCID, or PII.

关键词技巧

  • 先用 2-4 个核心英文词检索，查看结果后再逐步加字段标签。
  • 疾病和标准生物医学概念优先尝试精确 MeSH："Breast Neoplasms"[MeSH]。
  • 同义词可先用宽泛模式合并检索，再通过分面收窄。例如：breast cancer neoplasm carcinoma。
  • 需要高精度时限定题名：trastuzumab resistance[Title]。
  • PMID、DOI、PMCID 或 PII 可直接用于标识符检索。

Search Modes

Balanced
Default. Requires key terms and boosts phrase and identifier matches.
Strict
Requires all terms without phrase boosting.
Broad
Expands recall using OR plus minimum term matching.
Phrase
Requires phrase matching.

检索模式

均衡
默认模式。要求关键术语命中，同时对短语和标识符加权。
严格
要求全部术语命中，不启用短语加权。
宽泛
使用 OR 和最小匹配提高召回。
短语
要求短语匹配。

Compressed JSON Export

The web page exports the current query as .json.gz. Fixed-size exports keep the current result ordering. The “Export all” option uses deep pagination and PMID de-duplication.

POST /export/search?max_records=1000&source=full
POST /export/search?all_records=true&source=full

The file contains metadata, the normalized request, total hit count, exported count, and item records.

压缩 JSON 导出

网页会将当前检索条件导出为 .json.gz。固定条数导出保持当前结果排序；“全量导出”使用深分页并按 PMID 去重。

POST /export/search?max_records=1000&source=full
POST /export/search?all_records=true&source=full

文件包含元数据、规范化请求、总命中数、导出条数和文献记录。

PMC Full Text

When the PMC index exists, the result toolbar shows how many PMC full-text records correspond to the current PubMed query. Matching uses PMID and PMCID links.

POST /pmc/fulltext/count
POST /export/pmc-fulltext?source=full
GET /pmc/articles/PMC12823335

PMC 全文

当本地已建立 PMC 索引时，结果工具栏会显示当前 PubMed 检索对应的 PMC 全文数量。匹配依据 PMID 和 PMCID 关联。

POST /pmc/fulltext/count
POST /export/pmc-fulltext?source=full
GET /pmc/articles/PMC12823335

Retraction Status

Imports tag retracted publications, retraction notices, and expressions of concern from PubMed publication types and correction relationships.

POST /search/advanced
{"filters":[{"field":"publication_status_normalized","values":["retracted"]}]}

撤稿状态

导入时会根据 PubMed 出版类型和更正关系标记已撤稿文献、撤稿公告和表达关注记录。

POST /search/advanced
{"filters":[{"field":"publication_status_normalized","values":["retracted"]}]}
